# 📈 Metrics & KPIs

## North Star Metric
**Outfit Save Rate** — % of users who save at least one recommended outfit

## Key Metrics

| Metric | Definition | Target | Current | Status |
|--------|-----------|--------|---------|--------|
| Quiz Completion Rate | % of users who finish the full style quiz | >70% | — | ⏳ |
| Recommendation Satisfaction | User rating of outfit suggestions (1-5) | >4.0 | — | ⏳ |
| Brand Discovery Rate | % of recommended brands user has never heard of | >60% | — | ⏳ |
| Color Match Accuracy | % of users who agree the colors suit them | >80% | — | ⏳ |
| Time to First Recommendation | Seconds from start to seeing outfits | <90s | — | ⏳ |
| Return Rate | % of users who come back within 7 days | >25% | — | ⏳ |

## Funnel
```
Landing → Start Quiz → Complete Quiz → View Recommendations → Save Outfit → Return Visit
```
